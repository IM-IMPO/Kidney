## **Overview**

This project is a web-based Kidney Risk Assessment application designed to help users evaluate potential kidney-related risks through lifestyle questionnaires, laboratory input, and patient profiling. The goal is to make the interface clean, responsive, and easy to use across all device types.

I am opening this project for **community feedback**, suggestions, and possible improvements.
